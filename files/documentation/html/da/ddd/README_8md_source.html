1 # Onion Routing
2 
3 A final project for the Gvahim program. this project offers a system that creates anonimizer based on socks5 protocol
4 (rfc 1928 - https://www.ietf.org/rfc/rfc1928.txt).
5 The program is based on different nodes and a registry contains name, ip, and secret key of each node.
6 Registry is HTTP server, allowing register and unregister services.
7 Each node is a listener which registers itself to registry and listens to a chosen port.
8 
9 The program works the following way:
10 The client (browser) connects to the first node via socks5. Once connection is recieved the first node will look up the
11 registry for three additional random nodes.
12 The first node will establish socks with the second node encrypted using second node's key.
13 The first node will establish socks with the third node using third node's key.
14 The first node will establish socks with the fourth node using fourth node's key.
15 Once last connection is established the first node will forward the original socks connection from the client to
16 the fourth node encrypted by fourth node's key.
17 In practice it will be as if browser connected directly to the fourth node via socks, thus creating an anonimizer.

20 ## Getting Started
21 
22 These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.
23 
24 ### Prerequisites
25 
26 In order to run the project there are some requirments:
27 ```
28 1) Download and install Python 2.7 (https://www.python.org/download/releases/2.7/)
29 2) Download this repository and extract the zip.
30 3) Download and install the latest version of Firefox (https://www.mozilla.org/en-US/firefox/)
31 4) Change Firefox settings to connect via Socks5 to your local ip and port 1080.
32 5) Change the address in all configuration files to your ip.
33 ```
34 
35 ### Execution
36 
37 To execute the project open Command Prompt (or Terminal on Linux):
38 Reach the parent folder of this project:
39 ```
40 cd [location of onion_routing]
41 ```
42 Running Registry:
43 ```
44 python -m registry [args]
45 ```
46 Running Regular Node:
47 ```
48 python -m node_server [args]
49 ```
50 Running The first node:
51 ```
52 python -m node_client [args]
53 ```
54 
55 ### Arguments
56 
57 The node_server requires bind port, so excecution will be:
58 ```
59 python -m node_server --bind-port 2080
60 ```
61 Note: Other arguments for both node_server and registry_node are optional, see --help or -h for help.
62 
63 
64 ### Graphical Interface
65 
66 There is no graphical interface as part of the main program.
67 In order too enter the GUI in your prefered browser (not Firefox as Firefox should be working with socks5)
68 type:
69 ```
70 (your_ip):8080/statistics
71 ```
72 This will open the main page where statistics about the program may be seen.
73 
74 Note: Unless registry is running GUI won't work as it uses the HTTP server.
