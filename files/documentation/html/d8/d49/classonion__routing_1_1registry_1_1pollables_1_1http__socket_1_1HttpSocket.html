Get events for poller.

(int) events to register for poller.

POLLIN when:

  • _state is ACTIVE.
  • _buffer is not full.
  • _machine_state is less then RECV_CONTENT. POLLOUT when:
  • _buffer is not empty.
  • SEND_STATUS <= _machine_state <= SEND_CONTENT
